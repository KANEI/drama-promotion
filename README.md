# ベイズ統計モデリングによる公演動員数予測

## 概要
- このプロジェクトは、過去の公演データを用いて、将来の公演における動員数をベイズ統計モデリング（ロジスティック回帰）を用いて予測するものです。
- PythonのライブラリであるPyMCを使用し、マルコフ連鎖モンテカルロ法（MCMC）によってパラメータの事後分布を推定します。最終的に、特定の条件下（公演数、有料/無料など）での集客率と具体的な動員数を予測し、その結果をテキストファイルに出力します。

## 主な機能
1. **データ集計**: 過去データから企画ごとの「平均キャパ数」「平均予約数」「予約率」を自動計算。
2. **可視化**: 
   - データのヒストグラム
   - 企画ごとの推移プロット（キャパ数・予約数・予約率の2軸グラフ）
   - 自己相関関数（ACF）・偏自己相関関数（PACF）プロット
3. **MCMCモデリング**: PyMCを用いた二項分布＋ロジットリンク関数によるパラメータ推定。
4. **予測・レポート出力**: 指定した「ステージ数」「キャパシティ」に基づく動員数予測をテキストファイルとして出力。

## 1. データ概要
使用データ：`過去の本企画集客2.csv`

| カラム名 | 説明 |
| :--- | :--- |
| **企画** | 企画名称（時系列順に並んでいることを想定） |
| **キャパ** | 会場の収容人数（キャパシティ） |

## 2. 実行方法
1. `data` フォルダを作成し、対象のCSVファイルを配置します。
2. スクリプト内の予測パラメータ（`num_stages`, `capacity`）を必要に応じて変更します。
```python
num_stages = 5     # 予測したい公演のステージ数
capacity = 35      # 予測したい公演の1ステージあたりのキャパシティ
```
3. 以下のコマンドでスクリプトを実行します。
```python
python baysian_model.py
```
4. 処理が完了すると、`outputs` フォルダ内にグラフ画像と予測レポート（テキスト）が生成されます。

## 3.モデルの説明
このプロジェクトで使用しているベイズモデルの構造は、以下の図の通りです。

```math
X_{i} \sim Binormal(N_{i}, p_{i}) \
p_{i} = logit^{-1}(\eta_{i}) \
\eta_{i} = \beta_{0} + \beta_{1} * Y_{i} \
\beta_{0}, \beta_{1} \sim iid. Normal(\mu, \sigma) 
```

ただし $N_{i}$ は公演ごとの合計キャパ数、$X_{i}$は公演ごとの合計予約数、$Y_{i}$は公演数を表します。
